## 概要

CC-RetroEffect は、**AviUtl ExEdit2** 向けの高品質なレトロ映像エフェクトパッケージです。  
平成初期（1990年代前半）の **VHSビデオ・ブラウン管テレビ**の映像特性を忠実に再現した12種のエフェクトを、個別のスクリプトとして提供します。

エフェクトはそれぞれ独立したファイルとして動作し、自由に組み合わせてレイヤーに適用できます。  
HLSL シェーダー（GPU処理）により、FHD 解像度でもリアルタイムプレビューに対応します。

---

## 特徴

- 🎞️ **本物志向** — 実際の VHS 信号処理・ブラウン管の物理特性に基づいたエフェクト
- 🧩 **モジュラー設計** — 12 種のエフェクトを自由に組み合わせ可能
- ⚡ **GPU 処理** — HLSL シェーダーによる高速処理でリアルタイムプレビューに対応
- 📐 **解像度無制限** — SD から 8K 以上まで、解像度を問わず同一の見た目を維持
- 🎛️ **豊富なプリセット** — よく使う設定をワンクリックで適用できるエイリアスを同梱

## インストール

1. リリースページからスクリプト一式をダウンロード
2. `CC-RetroEffect` フォルダごと AviUtl2 の **`Script`** フォルダ内に配置

```
AviUtl2/
└── Script/
    └── CC-RetroEffect/
        ├── 01.コマ落ち＠CC-RE.anm2
        ├── 02.色調補正＠CC-RE.anm2
        ├── 03.ブロックノイズ＠CC-RE.anm2
        ├── 04.発光＠CC-RE.anm2
        ├── 05.色ずれ＠CC-RE.anm2
        ├── 06.カラーノイズ＠CC-RE.anm2
        ├── 07.ヘッドスイッチングノイズ＠CC-RE.anm2
        ├── 08.ドロップアウト＠CC-RE.anm2
        ├── 09.ジッター＠CC-RE.anm2
        ├── 10.画質劣化＠CC-RE.anm2
        ├── 11.ピラーボックス＠CC-RE.anm2
        └── 12.走査線＠CC-RE.anm2
```

3. AviUtl2 を起動（または再起動）

インストール後、タイムライン上のオブジェクトにフィルタオブジェクトとして追加できます。  
エフェクトは **「CC-RetroEffect」** のカテゴリ以下に表示されます。

## 使い方

### 基本的な使い方

1. タイムライン上を右クリック
2. **「フィルタオブジェクトを追加」** から `CC-RetroEffect` のカテゴリを選択
3. 01～12のエフェクトをクリックして追加
4. 追加されたエフェクトのパラメータを調整

### 推奨適用順序

複数のエフェクトを組み合わせる場合、以下の順序での適用を推奨します。
もちろん一つのエフェクト単体で動きます。
（上が先、下が後に処理されます）

```
 1. コマ落ち           — フレーム制御は最初に適用
 2. 色調補正           — 素材の色味を調整
 3. ブロックノイズ     — デジタル劣化を加える
 4. 発光               — ブラウン管のにじみを加える
 5. 色ずれ             — YIQ 信号の色滲みを加える
 6. カラーノイズ       — 磁気テープ由来のノイズを加える
 7. ヘッドスイッチングノイズ — 下端ノイズを加える
 8. ドロップアウト     — テープ傷を加える
 9. ジッター           — 水平揺れ・歪みを加える
10. 画質劣化           — ぼかし/シャープ処理
11. ピラーボックス     — 黒帯を追加
12. 走査線             — CRT 表示の最終処理
```

エイリアスとして以下のプリセットが付属しています。

| プリセット名 | 概要 |
|-------------|------|
| **VHS Standard** | 標準的な VHS 録画・再生の見た目 |

| **VHS SP** | SP 録画モード（高画質寄り） |

| **ブラウン管TV** | ブラウン管テレビの表示特性のみを再現 |

プリセットは `Alias/CC-RetroEffect/` フォルダの `.exa2` ファイルから適用できます。
